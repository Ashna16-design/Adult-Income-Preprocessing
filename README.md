# Adult-Income-Preprocessing
Data preprocessing for Adult Income dataset
## Tasks Performed
- Indetified categorical and numerical features
- Applied Label Encoding to ordered features
- Applied One-Hot encoding to unordered features
- Scaled numerical features using StandardScaler
- Prepared dataset for machine learning models.

  ## tools Used
  -Python'
  - Pandas
  - Scikit-learn
  - Google Colab

 ## Output
 The dataset is fully preprocessed and ready for machine learning.
